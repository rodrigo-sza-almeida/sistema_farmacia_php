
* Projeto desenvolvido em sala aula teve como objetivo implementar um sistema básico de controle de medicamentos utilizados em uma farmácia hospitalar.

- O front-end foi desenvolvido com HTML, CSS e programação PHP para saída de inputs, e o menu com implementação de programação Javascript para produzir alguns efeitos.
- O back-end foi utilizado a linguagem PHP com integração ao banco de dados MySql.
- O sistemante basicamente utiliza operações baseadas no conceito de CRUD para alterações no banco de dados.
